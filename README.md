# HFT-BO-WEB

> 基于 Vue 2.x 和 View Design 构建的后台管理系统，具备完善的模块化开发结构与构建优化配置，适用于高频交易后台、运营管理平台等场景。

## 📦 项目简介

`hft-bo-web` 是一个使用 Vue 2.7 构建的后台管理系统前端项目，采用现代前端开发理念，集成 Vue Router、Vuex 进行路由和状态管理，使用
View Design（iView）组件库构建 UI，同时融合 axios、moment、lodash 等优秀工具库，配合 Webpack 5 进行构建优化，支持 gzip
压缩、代码分割、版本控制等特性。

---

## 🧱 核心技术栈

| 类型      | 技术                                      | 描述                               |

| 请求库     | `axios`                                 | 数据请求、拦截、错误处理统一封装                 |
| 日期处理    | `moment`                                | 日期/时间处理工具                        |
| 工具库     | `lodash`                                | 数据处理、函数节流、防抖等工具函数                |

| 引导组件    | `intro.js`                              | 新手引导、页面提示指引                      |
| 预处理器    | `less` + `less-loader`                  | 样式模块化与复用                         |
| 打包工具    | `webpack@5.x`                           | 模块打包与构建优化                        |
| 构建脚本    | `vue-cli-service`                       | Vue CLI 提供构建命令支持                 |
| 代码校验    | `eslint` + `eslint-plugin-vue`          | 保持代码风格统一性                        |
| 构建分析    | `webpack-bundle-analyzer`               | 可视化包大小分析                         |
| 版本控制    | 自定义 `update-version.js`                 | 自动更新版本号，支持 CI/CD                 |
| Gzip 压缩 | `compression-webpack-plugin`            | 减少静态资源体积                         |
| CSS 压缩  | `cssnano`, `mini-css-extract-plugin`    | 提高构建效率和加载性能                      |

## 📁 推荐项目结构

```bash
hft-bo-web/
├── public/                  # 静态资源目录（如 index.html）
├── src/
│   ├── assets/              # 图片、样式资源
│   ├── components/          # 通用组件
│   ├── router/              # 路由配置
│   ├── store/               # Vuex 状态管理
│   ├── utils/               # 工具函数，如请求封装、加密函数等
│   ├── pages/               # 页面组件
│   ├── config/              # 全局配置（如环境变量、接口地址）
│   ├── App.vue              # 根组件
│   └── main.js              # 项目入口文件
├── update-version.js        # 自定义版本更新脚本
├── vue.config.js            # 构建配置（压缩、分析等）
└── package.json             # 项目信息与依赖
```

## 安装与运行步骤

1. **克隆项目到本地：**

   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **安装依赖：**

   确保已安装 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)，然后运行：

   ```bash
   npm install
   ```
3. **启动开发服务器：**

   ```bash
   npm run start
   ```
   访问 [http://localhost:8080](http://localhost:8080) 查看项目运行效果。
4. **更新版本：**

   ```bash
   npm run version x.x.x
   ```
